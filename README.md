# MQ2AutoLogin
MQ2AutoLogin will let you automatically log in characters and accounts simply by starting EQ with MQ2 and this plugin loaded.  [More info](http://www.macroquest2.com/phpBB3/viewtopic.php?f=50&t=16427)
